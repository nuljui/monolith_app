# Monolith App

A modern, high-performance web application for artifact management and project collaboration.

## Features

*   **Artifact Management**: Upload, view, and organize digital artifacts.
*   **Project Organization**: Group artifacts into projects with team members.
*   **Search**: Advanced search capabilities with visual search support.
*   **Profile**: manage your collections and settings.

## Getting Started

### Prerequisites

*   Node.js (v18+)
*   npm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

## Tech Stack

*   React
*   TypeScript
*   Tailwind CSS
*   Vite
*   Lucide React
